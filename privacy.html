1. Pendahuluan

Marstorekita menghormati privasi pengguna dan berkomitmen untuk melindungi data pribadi Anda. Kebijakan ini menjelaskan bagaimana kami mengumpulkan, menggunakan, menyimpan, dan melindungi informasi Anda saat menggunakan layanan top-up Robux dan Gamepass Roblox.

2. Informasi yang Kami Kumpulkan

  • Data Pribadi: Nama lengkap, alamat email, nomor telepon, ID Roblox, dan username Roblox
  • Data Transaksi: Riwayat pembelian, nominal top-up, metode pembayaran, dan detail transaksi
  • Data Teknis: Alamat IP, perangkat, browser, sistem operasi, dan cookie
  • Data Komunikasi: Chat, email, dan pesan yang Anda kirimkan ke tim dukungan

3. Penggunaan Informasi

  • Memproses transaksi top-up Robux dan pembelian Gamepass
  • Mengirimkan konfirmasi pembayaran dan notifikasi penting
  • Meningkatkan layanan berdasarkan feedback pengguna
  • Mencegah penipuan dan aktivitas tidak sah
  • Menyediakan dukungan pelanggan yang efektif

4. Penyimpanan dan Keamanan Data

  • Semua data disimpan di server aman dengan enkripsi SSL
  • Akses data dibatasi hanya untuk personel yang berwenang
  • Sistem keamanan bertingkat melindungi data dari akses tidak sah
  • Backup data terenkripsi dilakukan secara berkala
  • Tidak menyimpan detail kartu kredit (proses pembayaran melalui payment gateway)

5. Berbagi Informasi

Kami tidak menjual, menyewakan, atau membagikan data pribadi Anda kepada pihak ketiga, kecuali:

  • Dengan persetujuan eksplisit Anda
  • Untuk memenuhi kebutuhan hukum atau perintah pengadilan
  • Dengan penyedia layanan tepercaya (payment gateway, email service) yang terikat kerahasiaan data

6. Hak Pengguna

Anda berhak untuk:

  • Mengakses data pribadi yang kami simpan
  • Memperbarui atau memperbaiki informasi akun
  • Menghapus akun dan data pribadi Anda
  • Menolak penggunaan cookie (pengalaman website mungkin terbatas)
  • Mengajukan pertanyaan tentang penggunaan data Anda

7. Cookie

Kami menggunakan cookie untuk:

  • Meningkatkan pengalaman pengguna di website
  • Mengingat preferensi dan sesi login
  • Analisis pengunjung (Google Analytics)
  • Menampilkan promo yang relevan

Anda dapat mengatur browser untuk menolak cookie, tetapi beberapa fitur website mungkin tidak berfungsi dengan baik.

8. Retensi Data

  • Data transaksi disimpan minimal 2 tahun untuk keperluan pajak dan dukungan
  • Data pribadi akan dihapus secara permanen setelah Anda menghapus akun
  • Data cookie akan dihapus saat Anda keluar dari sesi atau membersihkan browser

9. Keamanan Roblox

  • Kami tidak menyimpan password Roblox Anda
  • Kami tidak meminta kata sandi Roblox untuk top-up (hanya ID/username)
  • Integrasi resmi dengan sistem Roblox untuk pengiriman Robux
  • Proses top-up dilakukan melalui API resmi Roblox
